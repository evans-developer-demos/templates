# Next.js Starter

Welcome to your new Next.js project! 

## Developing Locally

To get started with local development install [nvm](https://github.com/nvm-sh/nvm/blob/master/README.md)
then run the following commands:

```bash
git clone $REPO_URL nextjs-starter

cd nextjs-starter

nvm use

npm i

npm run dev
```
